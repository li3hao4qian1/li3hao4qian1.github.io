给出两个数 和 。由你来构造两个数组 ，满足如下条件：

. 两个数组的长度均为 。

. 两个数组中的数字都在 和 之间。

数组保证后一个数字小于等于前一个数字。

. 对于 到 的所有位置 ，都满足 。

问有多少种构造数组的方案，由于结果很大，只需要输出对 取模的结果。

看到这道题的我，当场开心的打了一个DFS.
DFS代码(50pts)
//注:下面这行注释是我考试时候写的，表达了作者的思乡之情。

#include<bits/stdc++.h>
using namespace std;
int n,m,a[15],b[15],ans,cnt,sum;
#define P 1000000007
void dfs(int dep){
    if(dep<=0) cnt++;
    else{
        for(int i=1;i<=a[dep+1];i++){
            a[dep]=i;
            dfs(dep-1);
            a[dep]=0;
        }
    }
}
void dfs1(int dep){
    if(dep<=0) sum++;
    else{
        for(int i=n;i>=b[dep+1];i--){
            b[dep]=i;
            dfs1(dep-1);
            b[dep]=0;
        }
    }
}

                a[m]=0;
                b[m]=j;
                dfs1(m-1);b[m]=0;
//                cout<<ans<<cnt<<sum<<endl; 
                ans+=(cnt*sum)%P;
            }

思路屎粪清晰，因为a数组保证后一个数字大于等于前一个数字,
而b数组保证后一个数字小于等于前一个数字,
但是对于1到m的所有位置i，都满足a[i]≤b[i]。
也就是说，只需要考虑a[m]≤b[m]!
所以，只需要枚举所有a[m]≤b[m]，再逐个枚举a数组和b数组，最后方案数相乘再相加。
正解也是一样的思路。先画一个小图:
				a[m]≤a[m-1]≤...≤a[3]≤a[2]≤a[1]
				b[1]≤b[2]≤b[3]≤...≤b[m-1]≤b[m]
因为a[i]≤b[i],所以a[1]也≤b[1]。
因此，可以把下面一行移上去：
				a[m]≤a[m-1]≤...≤a[3]≤a[2]≤a[1]≤
				
				b[1]≤b[2]≤b[3]≤...≤b[m-1]≤b[m]


好样的CSS 这样，我们就可以对两个数组同时进行DP。
设立状态dp[i][j]表示填第i个数，上一个填了j的方案数。
状态转移方程：dp[i][j]=dp[i-1][j~n]。
初始dp[1][0~n]=1。
可以使用前缀和从O(n²m)优化为O(nm)。不过没必要。
AC代码
#include<bits/stdc++.h>
using namespace std;
#define P 1000000007
int n,m,dp[25][1005],ans;
int main(){
    cin>>n>>m;
    for(int i=1;i<=n;i++) dp[1][i]=1;
    for(int i=2;i<=2*m;i++){
        for(int j=1;j<=n;j++){
            for(int k=1;k<=j;k++){//这里状态转移方程有所不同，
                dp[i][j]+=dp[i-1][k];//因为我写的代码实际上
                dp[i][j]%=P;//把上面的移下来，所以是[1~j]。
            }
        }
    }
    for(int i=1;i<=n;i++){
        ans+=dp[2*m][i];
        ans%=P;
    }
    cout<<ans;
    return 0;
}

题目描述

路上有 个粮仓，分别位于 位置，每个粮仓保存了 个粮食。

两个人在玩抢粮食的游戏，其中小 已经将他的 个士兵，分配到了位置 到 的位置。

现在轮到小 放置他的 个士兵了，游戏规则如下：

. 小 的士兵位置不能与小 的士兵重合，但可以直接放到粮仓上。

. 如果小 的士兵距离某个粮仓更近，则粮仓中的所有粮食归小 所有，否则归小 （相等也是归小 ）

问小 最多能够获得多少粮食。

直接输出36可以获得5pts。 可以使用贪心算法。
对于每一个粮仓，都有两个最远的可以放置小兵的位置。
				敌 远             仓             远                       敌
				------------------------------------------------------------
我们可以使用二分查找决定这两个点的敌兵的位置和方向。
使用数组存储这些数据，排序再取前n位，相加就可以得到答案。
AC代码
#include<bits/stdc++.h>
using namespace std;
#define int long long
int k,n,m,ans,p[200005],a[200005],d[200005],an[600005];
signed main(){
    cin>>k>>m>>n;
    for(int i=1;i<=k;i++) cin>>p[i]>>a[i];
    for(int i=1;i<=m;i++) cin>>d[i];
    sort(d+1,d+m+1);
    for(int i=1;i<=k;i++){
        int pos=lower_bound(d+1,d+m+1,p[i])-d;
        int cur=0;
        if(pos>1&&p[i]-d[pos-1]<d[pos]-p[i]) pos--,cur=1;
        an[pos*2+cur]+=a[i];
    }
    sort(an+1,an+m*2+3,greater<int>());
    for(int i=1;i<=n;i++) ans+=an[i];
    cout<<ans;
    return 0;
}
